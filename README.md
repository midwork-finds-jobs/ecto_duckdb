# ecto adapter for duckdbex

This repo is initial starter point for Claude Code.

It implemeents ecto adapter to use duckdb. Ecto is an library to work with databases in elixir.

It contains DB_Connection behaviour for https://github.com/AlexR2D2/duckdbex

It contains ecto adapter for duckdb through the DB Connection and duckdbex.

It contains `./sample_phoenix` folder with a fully working phoenix example project using duckdb ecto adapter and local duckdb file.
